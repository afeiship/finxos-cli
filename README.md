# finxos-cli
> Cli tools for finxos.


## installation
```shell
git clone https://github.com/afeiship/finxos-cli.git
cd finxos-cli
npm install
npm link
```


## cmds
| api        | type | description                |
| ---------- | ---- | -------------------------- |
| tss-upload | -    | Upload dirs in current dir. |
